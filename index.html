3. Tìm hiểu và tạo Virtual host

3.1 Hosts file location

In windows 10 the host file is located at : c:\Windows\System32\Drivers\etc\hosts

3.2 Cách hoạt động của DNS

DN (Domain Name): tên miền . Ví dụ: localhost, google,…
DNS (Domain Name System): hệ thống tên miền là hệ thống chuyển đổi tên miền thành địa chỉ IP và ngược lại.
IP : địa chỉ IP.
* Một số địa chỉ IP thường gặp:
localhost: 127.0.0.1
Google.com.vn: 142.250.66.99
* Một số port phổ biến:

  1. HTTPS: 443 Bị ẩn
  2. HTTP: 80 Bị ẩn
  3. SSH: 22
  4. Mysql: 3306
  5. TCP: 21
  6. Redis: 6379

Cách hoạt động của DNS

Sau khi user nhập tên miền (ví dụ như example.com) vào trình duyệt web. DNS sẽ chuyển đổi tên miền thành IP mà máy tính có thể hiểu và kết nối thông qua tìm kiếm truy vấn vào tên miền đã được lưu trước đó (DNS local: độ ưu tiên cao)

3.3 Tạo virtual host thủ công
